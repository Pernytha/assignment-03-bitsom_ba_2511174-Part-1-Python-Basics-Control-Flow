# Student Grade Tracker

This is a simple Python program that manages student data and calculates grades.

## What the program does
- Cleans student names and converts marks into a list.
- Shows each student's profile with roll number and marks.
- Calculates grades, total, and average for each student.
- Lets you add new subjects and marks.
- Prints a class summary with pass/fail, topper, and class average.
- Performs some basic string operations on an essay.

## How to run
1. Open the file `part1_grade_tracker.py` in Python or VS Code.
2. Run the program.
3. Follow the instructions on the screen to enter any new subjects or marks.
